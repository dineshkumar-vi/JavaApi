10. src/main/java/com/example/controllor/PriceController.java
    - REST API controller at /api/prices
    - 12 endpoints:
      * POST /upload-pdf - Upload PDF and extract prices
      * POST / - Create price manually
      * GET / - Get all prices
      * GET /{id} - Get price by ID
      * PUT /{id} - Update price
      * DELETE /{id} - Delete price
      * GET /search/product-name
      * GET /search/product-code
      * GET /search/category
      * GET /search/supplier
      * GET /search/price-range
      * GET /search/keyword
    - Comprehensive error handling
    - File validation (PDF only, max 10MB)
    - CORS enabled

UTILITY LAYER:
11. src/main/java/com/example/util/PdfPriceExtractor.java
    - PDF text extraction using Apache PDFBox
    - Regex-based pattern matching for:
      * Prices: $99.99, £49.99, €79.99, ¥1,000, ₹999.00
      * Currency symbols and codes
      * Product codes: SKU, Code, Item patterns
      * Product names
    - Page-by-page processing
    - Context-aware extraction
